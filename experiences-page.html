My Experiences

My favorite part of being part of the Computer Science program is getting to create. In the time I have been in the program, I have made several projects I am proud of:

Board Game

This is a board game I made as the final project for my intermediate programming class in the spring semester of 2023. The game works similarly to chutes and ladders where if you land on a blue space, your player moves forward. If the player lands on a red space, the player moves backwards. I made the project in Java using NetBeans.

About Me App

In my Mobile Apps Development class, we made mobile apps using NativeScript. These apps are just basic about me apps with describing information using basic app development elements. The apps work on our phones using the NativeScript Preview App. The app changes between pages and each page uses a different component. There are ListViews, Buttons, Forms, Images, and Videos.

This is the prototype design from Figma, the wireframes created before the project was coded, and a video showing the project is working in the NativeScript preview app.

My Portfolio

For my portfolio project, I decided to create a static HTML page. The portfolio meets several specifications since it is also a class project, but it still showcases my work, community involvement, and my resume. Before I built it using HTML and CSS, I made the wireframes using Figma. The portfolio project was originally built using StackBlitz, but then I created a GitHub Repository and finished the project in Visual Studio Code. When I reached a point that it was mostly finished, I deployed the portfolio using Render. Every time I push to GitHub, it redeploys automatically ensuring it is always up to date.
